## Description

Survoon API

## Tech Stack

**Server:** NestJS

**DB:** Postgres


## Installation

Clone the project

```bash
  git clone https://github.com/theoblin/Survoon-API.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```




## Run Locally

Postgres + dpage/pgadmin4 start

```bash
  docker compose up
```
